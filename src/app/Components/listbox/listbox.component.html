TurboNG - ListBox Overview

A powerful and intuitive Angular component for dual-list selection. This library features a dual-sided listbox interface, allowing users to seamlessly move items between the left and right panels. Ideal for scenarios like permission management, or item transfers, it offers a clean design, smooth animations. With built-in support for selection, and responsive layouts, this component enhances productivity and user experience in any Angular application.
