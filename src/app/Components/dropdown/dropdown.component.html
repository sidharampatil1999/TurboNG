TurboNG - DropDown Overview

The TurboNG Dropdown library is a lightweight, flexible, and reusable Angular component designed to simplify the implementation of dropdown menus across your application. Built with scalability and customization in mind, this library allows developers to easily integrate feature-rich dropdowns with minimal setup

2. Add below component tag into .html file

● [ListItems] accept the array.
● 'ProvidedArray' is an array which declared in component.ts file.
● '(GetSelectedValue)' emits the selected value from dropdown.
● User can create a method like 'getDropdownValue($event)' to capture emitted value.
● DefaultValue='India' user can give prepopulated default value.
● class="dropdowncss" user can apply css to modify width & height.
