TurboNG - Radio Button Overview

A lightweight and customizable Angular component for creating radio button groups with enhanced flexibility. This library supports labels, enabled/disabled states, and seamless integration into forms and UI layouts. Designed for accessibility and responsiveness, it allows developers to easily implement intuitive selection controls with consistent styling and behavior across devices.
The next update will introduce label positioning options (left, right, top, bottom) and enhanced design themes, giving developers even more control over layout and styling.
